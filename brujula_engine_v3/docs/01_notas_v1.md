# Notas de Brújula Engine v1

Esta versión es deliberadamente simple. Su objetivo no es ser correcta en términos estadísticos, sino crear la primera base ejecutable para experimentar con escenarios.

Principios incorporados:
- Lenguaje humano.
- Indicadores con nombres claros.
- No decidir por la persona.
- Evitar que el dinero sea el único criterio.
