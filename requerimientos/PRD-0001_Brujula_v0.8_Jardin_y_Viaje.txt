# BRÚJULA
# PRD-0001
# Versión 0.8 — Separación de Modos: Jardín + Viaje

Estado: Aprobado para desarrollo

## 1. Visión
Brújula se divide en dos experiencias:

- 🌿 Modo Jardín: cultivar bienestar mediante pequeños rituales cotidianos.
- ✈️ Modo Viaje: calcular la mejor ruta para alcanzar un sueño o meta importante.

Ambos utilizan un único LifeProfile.

## 2. Problema
La UI actual mezcla bienestar, simulación y planificación en una sola pantalla.
Los indicadores no explican claramente qué hacer después.

## 3. Objetivos
- Separar presente y futuro.
- Hacer comprensible cada modo.
- Reutilizar LifeProfile.
- Simplificar la lectura de resultados.

## 4. Pantalla inicial

¿Qué quieres hacer hoy?

🌿 Cuidar mi Jardín
"Pequeños actos para sentirte mejor hoy."

✈️ Planificar un Viaje
"Explora distintos caminos para alcanzar un sueño."

## 5. Modo Jardín

Pregunta:
¿Qué necesita mi vida hoy?

Funciones MVP:
- Check-in diario.
- Estado del jardín.
- Ritual recomendado.
- Registro posterior (¿lo hiciste?, ¿cómo te sentiste?).

Check-in:
- Estado de ánimo
- Energía
- Dolor
- Sueño
- Tiempo disponible
- Necesidad principal
- Nota opcional

Indicadores:
🌿 Energía
🌸 Serenidad
🌳 Salud
🏡 Relaciones
🎨 Creatividad

Ejemplos de rituales:
☕ Preparar una bebida caliente.
🌳 Caminar 15 minutos.
📖 Escribir una página.
📷 Tomar una fotografía.
💤 Dormir antes.

## 6. Modo Viaje

Pregunta:
¿Cuál es el mejor camino para llegar a mi sueño?

Entradas:
- LifeProfile
- Escenario
- Restricciones

Resultado:
- Destino
- Ruta recomendada
- Probabilidad de éxito
- Tiempo estimado
- Esfuerzo requerido
- Principal ventaja
- Principal riesgo
- Historia del camino
- Carta de Sue
- Rituales relacionados
- Detalle avanzado colapsable

## 7. LifeProfile

Fuente única de:
- salud
- finanzas
- valores
- sueños
- preferencias
- restricciones

## 8. Arquitectura

GardenEngine
- check-in
- rituales
- evolución diaria

JourneyEngine
- simulaciones
- escenarios
- comparación

Servicios compartidos:
- LifeProfile
- Sue
- Knowledge Library

## 9. Cambios UI

Modo Jardín:
Estado actual + rituales.

Modo Viaje:
Destino → Ruta → Probabilidad → Tiempo → Riesgos → Hitos.
Las gráficas pasan a segundo plano.

## 10. Criterios de aceptación

Modo Jardín
✓ Registrar estado.
✓ Recibir rituales personalizados.
✓ Evaluar resultado.

Modo Viaje
✓ Crear escenario.
✓ Obtener ruta clara.
✓ Comprender ventajas y riesgos en menos de 30 segundos.

Generales
✓ Ambos modos usan el mismo LifeProfile.
✓ El usuario entiende claramente la diferencia entre ambos.
